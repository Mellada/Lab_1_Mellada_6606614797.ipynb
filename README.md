# Lab_1_Mellada_6606614797.ipynb
Spatial analysis of public bus stations in Ubon Ratchathani using GeoPandas and Folium. This project demonstrates buffer creation, spatial join, and impact area calculation with interactive web mapping visualization.

## 1. บทนำ (Introduction)
โปรเจกต์นี้เป็นการวิเคราะห์เชิงพื้นที่ (Spatial Analysis) เพื่อประเมินขอบเขตการให้บริการ (Service Area) ของสถานีขนส่งผู้โดยสารภายในจังหวัดอุบลราชธานี โดยใช้เทคนิคการสร้างพื้นที่กันชน (Buffer) ระยะ 5 กิโลเมตร เพื่อศึกษาความครอบคลุมของการเข้าถึงบริการขนส่งสาธารณะในแต่ละพื้นที่สำคัญของจังหวัด

## 2. วัตถุประสงค์ (Objectives)
* เพื่อจัดทำฐานข้อมูลตำแหน่งพิกัดของสถานีขนส่งผู้โดยสารในจังหวัดอุบลราชธานี
* เพื่อวิเคราะห์รัศมีการให้บริการในระยะ 5 กิโลเมตร โดยใช้ระบบพิกัด UTM (EPSG:32648) เพื่อความแม่นยำในการวัดระยะทาง
* เพื่อคำนวณพื้นที่ให้บริการจริง (Intersection) ที่ซ้อนทับอยู่ภายในขอบเขตการปกครองของจังหวัดอุบลราชธานี
* เพื่อนำเสนอข้อมูลในรูปแบบแผนที่เชิงโต้ตอบ (Interactive Map) ที่สามารถวิเคราะห์ข้อมูลรายสถานีได้

## 3. เครื่องมือและเทคโนโลยีที่ใช้ (Tech Stack)
* **ภาษา:** Python
* **คลังคำสั่ง (Libraries):**
    * `GeoPandas`: จัดการและวิเคราะห์ข้อมูลเชิงพื้นที่
    * `Pandas`: บริหารจัดการข้อมูลตาราง
    * `Folium`: สร้างแผนที่แบบ Interactive
    * `OSMnx`: ดึงข้อมูลขอบเขตจังหวัดจาก OpenStreetMap
    * `Shapely`: ประมวลผลรูปทรงเรขาคณิต (Point, Polygon, Buffer)

## 4. ขั้นตอนการดำเนินงาน (Methodology)
1. **เตรียมข้อมูล:** สร้างชุดข้อมูลตำแหน่งสถานีขนส่ง (Latitude/Longitude) และแปลงเป็น GeoDataFrame
2. **จัดการระบบพิกัด (CRS):** แปลงระบบพิกัดจาก WGS84 (EPSG:4326) เป็น UTM Zone 48N (EPSG:32648) เนื่องจากหน่วยเป็นเมตร เหมาะสำหรับการคำนวณระยะทางและพื้นที่
3. **วิเคราะห์เชิงพื้นที่:**
    * สร้างพื้นที่ Buffer ระยะ 5,000 เมตร รอบทุกสถานี
    * ดึงข้อมูลขอบเขตจังหวัดอุบลราชธานี และนำมาตัด (Intersection) กับพื้นที่ Buffer เพื่อหาพื้นที่ให้บริการที่อยู่ภายในจังหวัดจริง
4. **คำนวณพื้นที่:** คำนวณขนาดพื้นที่ให้บริการในหน่วยตารางกิโลเมตร
5. **แสดงผล:** สร้างแผนที่ผ่าน Folium โดยแสดงตำแหน่งสถานี (Markers) และขอบเขตพื้นที่ให้บริการ (Service Area)

## 5. สรุปผลการศึกษา (Results & Conclusion)
จากการวิเคราะห์เชิงพื้นที่ พบข้อสรุปที่สำคัญดังนี้:

* **ประสิทธิภาพการครอบคลุม:** ระบบสามารถระบุขอบเขตการให้บริการของสถานีขนส่งทั้ง 6 แห่งได้อย่างชัดเจน โดยสถานีในเขตเมืองมีการทับซ้อนของพื้นที่ให้บริการ ซึ่งบ่งบอกถึงความหนาแน่นของการเข้าถึงในศูนย์กลางจังหวัด
* **การคำนวณพื้นที่:** การใช้ระบบพิกัด UTM ทำให้สามารถคำนวณพื้นที่ให้บริการจริงได้อย่างแม่นยำ (เช่น สถานีหลักมีพื้นที่ให้บริการครอบคลุมพื้นที่ในจังหวัดประมาณ 78 ตร.กม.) ซึ่งพื้นที่เหล่านี้คือบริเวณที่ประชาชนสามารถเข้าถึงสถานีขนส่งได้สะดวกในระยะ 5 กิโลเมตร
* **การใช้งานจริง:** แผนที่ Interactive ช่วยให้ผู้บริหารจัดการเมืองหรือหน่วยงานขนส่ง สามารถมองเห็น "ช่องว่าง" (Gap Area) ที่บริการขนส่งสาธารณะยังเข้าไม่ถึงได้ชัดเจน ซึ่งเป็นประโยชน์ต่อการวางแผนขยายโครงข่ายคมนาคมในอนาคต
